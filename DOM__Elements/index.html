В этом уроке вы узнаете, как находить и получать доступ к HTML-элементам на HTML-странице

Поиск HTML-элемента по ID

"document.getElementById('*.*')"

Поиск HTML-элементов по имени тега

'document.getElementsByTagName(*.*)'

В этом примере выполняется поиск элемента с id="main", а затем поиск всех 'p' элементов внутри "main"

'const x = document.getElementById("main")'
"const y =x.getElementsByTagName('p')"

Поиск HTML-элементов по имени класса

"const classText = document.getElementByClassName('*.*')"

Поиск HTML-элементов с помощью селекторов CSS

const cssAll = document.querySelectorAll(.*)
