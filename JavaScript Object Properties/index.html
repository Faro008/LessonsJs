Свойства объекта JavaScript

Доступ к свойству с помощью выражения

const person = { firstName: "John", lastName : "Doe", age : 50 }; let x = "firstName"; let y = "age"; document.getElementById("demo").innerHTML = person[x] + " is " + person[y] + " years old.";

Доступ к свойству с помощью []

const person = { firstname: "John", lastname: "Doe", age: 50, eyecolor: "blue" }; document.getElementById("demo").innerHTML = person["firstname"] + " is " + person["age"] + " years old.";
