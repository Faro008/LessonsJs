Objects

JavaScript-объекты

const car = {type: Fiat, model: 500, color: white}; document.getElementById('intro').innerHTML = 'Модель машины: '+car.model+', цвет: '+car.color;

В этом примере создается пустой объект JavaScript, а затем добавляются 4 свойства:

const person = {}; person.name = 'John Doe'; person.age = 30; person.city = 'New York'; intro.innerHTML = 'Имя: '+person.name+', возраст: '+person.age+', город: '+person.city;

Использование нового ключевого слова

const person = new Object(); person.name = 'John Doe'; person.age = 30; person.city = 'New York'; intro.innerHTML = 'Имя: '+person.name+', возраст: '+person.age+', город: '+person.city;

Методы — это действия , которые можно выполнять над объектами.

Методы — это определения функций, хранящиеся в виде значений свойств

const person = { firstName: "John", lastName: "Doe", id: 5566, fullName: function() { return this.firstName + " " + this.lastName; } };
